# User Guide

Complete guide to using Task Manager application.

## Getting Started

### Creating an Account

1. Visit the application URL
2. Click **"Sign Up"**
3. Enter your name, email, and password
4. Click **"Create Account"**

### Logging In

1. Click **"Login"**
2. Enter your email and password
3. Click **"Sign In"**

---

## Managing Tasks

### Creating a Task

1. Click the **"+ Add Task"** button
2. Fill in the task details:
   - **Title** (required)
   - **Description** (optional)
   - **Priority** (Low, Medium, High)
   - **Due Date** (optional)
   - **Tags** (optional)
3. Click **"Create Task"**

### Editing a Task

1. Click on any task card
2. Modify the fields you want to change
3. Click **"Save Changes"**

### Completing a Task

- Click the checkbox on the task card
- Or open the task and click **"Mark as Complete"**

### Deleting a Task

1. Open the task
2. Click the **"Delete"** button
3. Confirm deletion

---

## Advanced Features

### Voice Input

1. Click the microphone icon in the task creation form
2. Say your task naturally:
   - "High priority meeting tomorrow at 3pm with work tag"
   - "Buy groceries due Friday"
3. The app will automatically parse and fill in the details

### Recurring Tasks

1. Create or edit a task
2. Enable **"Recurring"** option
3. Select frequency:
   - Daily
   - Weekly
   - Monthly
4. The task will automatically recreate after completion

### File Attachments

1. Open a task
2. Click **"Add Attachment"**
3. Select up to 5 files
4. Files are uploaded and linked to the task

### Task Sharing

1. Open a task
2. Click **"Share"** button
3. Search for users by email
4. Select users to share with
5. They can now view and edit the task

---

## Organization

### Using Tags

- Add tags when creating tasks
- Click on tags to filter tasks
- Use tags like: work, personal, urgent, shopping

### Filtering Tasks

Use the sidebar filters:
- **Status**: All, Pending, Completed
- **Priority**: All, High, Medium, Low
- **Tags**: Click any tag to filter
- **Date Range**: Filter by due date

### Searching Tasks

- Use the search bar in the navbar
- Search by title or description
- Results appear instantly

### Sorting Tasks

Click the sort dropdown:
- By Date Created
- By Due Date
- By Priority
- Alphabetically

## Gamification

### Earning Points

- Complete a task: +10 points
- Complete high priority task: +20 points
- Maintain streak: Bonus points

### Streaks

- Complete at least one task daily
- Your streak increases each consecutive day
- Streaks reset if you miss a day

### Badges

Unlock badges by:
- Completing 10, 50, 100 tasks
- Maintaining 7, 30, 100 day streaks
- Using all features

### Leaderboard

- View top users by points
- Compete with other users
- Track your ranking

## Import & Export

### Exporting Tasks

1. Click your profile icon
2. Select **"Export Tasks"**
3. Choose format:
   - CSV (for Excel)
   - JSON (for backup)
4. File downloads automatically

### Importing Tasks

1. Click your profile icon
2. Select **"Import Tasks"**
3. Choose your file (CSV or JSON)
4. Tasks are imported and appear in your list

## Offline Mode

### Using Offline

- The app works without internet
- Create and edit tasks offline
- Changes are saved locally

### Syncing

- When internet returns, changes sync automatically
- Conflicts are resolved automatically

### Installing as App

1. Click browser menu
2. Select **"Install Task Manager"**
3. App installs on your device
4. Launch like a native app

## Troubleshooting

### Tasks Not Syncing

1. Check internet connection
2. Refresh the page
3. Log out and log back in

### Attachments Not Uploading

- Check file size (max 5MB per file)
- Ensure supported file type
- Try a different browser

### Voice Input Not Working

- Allow microphone permissions
- Use Chrome or Edge browser
- Speak clearly and naturally
